

# 🤖 AI Phone Agent

A Siri-like AI agent that controls your device through natural language commands.

## 🚀 Quick Start

```bash
git clone https://github.com/Milan-Gautam/ai-phone-agent.git
cd ai-phone-agent
npm install
npm start
Open http://localhost:5000
```

## ✨ Features

🗣️ Natural language commands

🌐 Open any website

🔍 Search Google/YouTube

📱 Open system apps

💬 Chat-like interface

🔒 Local data storage

🆓 100% Free & Open Source
